Библиотека содержит: 
1. компонент отображения калькулятора в папке component
2. структуру и тестовые данные инфоблока калькулятора в папке iblock


Калькулятор поддерживает:
1. 
работу разделов как групп услуг
2.
расчет для каждой услуги по алгоритму: 

количество единиц заказываемой услуги 
- умножается на тариф за единицу услуги
- делится на количество расчетных периодов (если они отличны от 1 мес)
- умножается/делится/складывается/вычитается по выпбранным пунктам 5 корректирующих коэффициентов
 



Для значений корректирующего коэффициента можно указывать тип операции по шаблону "модификаторЗначение коэффициента".
 
 Список модификаторов:
 "*" - умножить
 "/" - разделить
 "+" - прибавить
 "-" - вычесть
 
 Если ничего не указано применяется модификатор "+".
 
 Пример.
 Запись "*1.2" означает умножение суммы услуги на 1.2





Внешний вид полей выбора корректирующих коэффициентов может быть модифицирован следующим образом:
 
В поля: "Название N корректирующего коэффициента" можно добавлять модификаторы внешнего вида по шаблону: "Название поля /модификатор"
 
 Список модификаторов:
 /l - выпадающий список
 /ml - выпадающий список с множественным выбором
 /с - чекбоксы (квадратики)
 /r - радиокнопки (кружочки)
 
 Если ничего не указано, то по умолчанию применяется модификатор /r.
 
 Пример. 
 Запись: "Степень риска /с" выведет весь список значений корректирующего коэффициента "Степень риска" в виде чекбоксов.
